# Contributing

Thanks for considering a contribution to Thought API.

This repository is an experimental open-source snapshot. Keep changes focused, include tests for behavior changes, and run the local verification commands before opening a pull request:

```sh
npm ci
npm run build
npm test
```

Do not submit secrets, non-public datasets, local databases, logs, uploads, production deployment configuration, or internal planning material.

For security issues, follow `SECURITY.md` instead of opening a detailed public issue.
